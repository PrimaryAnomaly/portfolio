## Problem

Characterizing multi-channel sensor arrays requires real-time visualization of analog signals alongside automated calibration routines. Commercial DAQ software lacks the flexibility to handle custom sensor protocols and calibration models.

## Approach

Built a Python application using PyQt for the GUI and PyQtGraph for real-time plotting. The backend interfaces with NI DAQ hardware via NI-DAQmx drivers. Implemented automated multi-point calibration with least-squares regression, data logging to HDF5, and configurable alarm thresholds.

## Result

Replaced a LabVIEW-based workflow that required manual data export and offline calibration. Reduced calibration time from 2 hours to 15 minutes. Used across three concurrent research projects in the lab.
